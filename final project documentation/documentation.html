About this Website

Fresh website is a shopping website that sells home appliances varying from small to large appliances, like TVs, cookers, microwaves, refrigerators, ACs and others, and it has full payment integration and customer login for adding a wishlist and contact information in case of purchasing them online, it also takes the order to the next level for delivery and cash collection.

Host Configuration and Theme creation

Starting is always hard but it gets harder by the time

  1. The first thing you need to do is to choose a good server and host for your website.

  2. Second thing you need to do is to choose a domain name for your website and reserve it, and activate the SSL certificate to allow integration with social media and payment methods.

  3. The Third thing you need to do is to choose a good theme that will work with your website imagination.

  4. Fourth thing is to install the theme and make sure it is responsive and adequate.

  5. Next thing you need to do is to install the theme.

  6. The website now is ready for some plugins to bring it to life!

Opening the Website

We have made quite a good user experience in the website, starting for the very beginning to the very end

  1. Upon opening the website, you get to browse the products and the varies categories with the offers.

  2. After choosing whatever you like, you can then start adding them to our responsive accumilative cart or to your responsive wishlist or even compare between 2 products to make it easier for you to choose between them.

  3. You can then view your cart and decide on what you need (if not all!) and what you will remove for now.

  4. You can now head to the Checkout after logging in and leaving you address information, to add your payment method and start placing the order to get a confirmation e-mail with the order number!

And that is how easy our website navigation is!

Backup


Oops! Your website is down and everything seems to be lost forever?! Don't think so, thanks to our backup plugin, you can now retrieve all the lost data, plugins, themes and more just to have a plan B in case anything goes astray!

String Locator

If you are looking for a specific line of code to adjust it or add anything to your website for a better layout or dynamism, that's a headache surely given the thousands of codes in the editors and files, but we can easily find any line of code, php, html, JS or even CSS with just a copy and paste command in our String Locator plugin, so no more wasting time searching for a needle in a haystack!

Important: Please take proper care not to edit anything in the main PHP files (wp-config.php, wp-content.php), without having a backup as anything might go wrong!

Filters
